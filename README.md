# Google route finder

App is created for get route direction on google map

Technologies Used:

* React
* Jest
* Enzymes
* ES 6
* GIT

## Prerequisite 
node
NPM
and 
yarn

## Installation

```
npm install
or
yarn install
```


Add a `.env` file with the following:

```
REACT_APP_GOOGLE_API_KEY = <api-key>
```

## Configuration 

```
REACT_APP_BASE_URL = <mock api url>
```


## Run Tests

```
npm test
or 
yarn test
```

## Run App in development mode

```
npm  start
or 
yarn start
```

## Build for production

```
npm run build
or 
yarn build
```
